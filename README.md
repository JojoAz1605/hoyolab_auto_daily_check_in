Voici un petit guide d'utilisation:
1. Installez python 3.9 ou 3.10 sur votre machine
2. Lancez le install_requirements.bat
3. Ouvrez le .env, et renseignez votre adresse mail ainsi que votre mot de passe, sauvergardez
4. Lancez genshin_daily.py ou honkai_daily.py
5. Un captcha va vous être soumis, faites le, et retournez sur le programme, vos récompenses devraient avoir été récupérées!
